# Ciclo While: Repetir hasta que me canse 🔄

segun el video  el ciclo `while` A diferencia del `for`, este se sigue ejecutando **mientras** una condición sea verdad ss com mientras haya sopa en el plato, sigue comiendo


**Mi ejemplo del video:**
```python
contador = 1
while contador <= 5:
    print("Voy por el número:", contador)
    contador += 1  # ¡Súper importante para que no sea infinito!

print("¡Terminé el ciclo!")
```
